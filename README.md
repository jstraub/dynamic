Simulation of some common dynamical systems that exhibit chaos.

The project depends on
[Pangolin](https://github.com/stevenlovegrove/Pangolin) for
visualization and
[Eigen](http://eigen.tuxfamily.org/index.php?title=Main_Page) for
numerics.
